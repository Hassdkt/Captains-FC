# Touch Juggling Update

The active race gameplay has been changed from camera detection to a finger-controlled avatar game.

## Included
- Touch controls for right foot, left foot, thighs, laces, inside, and outside.
- Progressive sequence support for combo challenges.
- Alternating-foot challenges require right/left rhythm.
- Wrong taps do not reset completed progress.
- Automatic finish when the target is reached.
- Solo and multiplayer race screens both use the touch game.
- Existing room, host, leaderboard, scoring, difficulty, and round systems are preserved.

## Run locally
```bash
npm install
npm run dev
```
